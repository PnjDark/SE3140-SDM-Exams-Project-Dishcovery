ğŸš€ Getting Started

Quick Start (5 minutes)

  1. Download and install Node.js 18+ and MySQL 8+
  2. Clone the project repository
  3. Run database setup: node database/setup.js
  4. Create admin user: node database/create-admin.js
  5. Start backend: cd server && npm run dev
  6. Start frontend: cd client && npm start
  7. Visit http://localhost:3000

Access Points

Frontend: http://localhost:3000
Backend API: http://localhost:5000
Admin Dashboard: http://localhost:3000/dashboard/admin

About Us: http://localhost:3000/about
